üë§ User Authentication APIs

POST /api/auth/user/register
Register a new user account. Creates a user with email and password authentication.
Request Body
Parameter Type Required Description
firstName String Yes User's first name
lastName String Yes User's last name
email String Yes User's email address (must be unique)
password String Yes Password (minimum 8 characters)
confirmPassword String Yes Password confirmation (must match password)
country String Optional User's country
role String Optional User role (default: "user")
subscription String Optional Subscription type: "free" or "premium" (default: "free")
Request Example
{ "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "password": "password123", "confirmPassword": "password123", "country": "Nepal", "role": "user", "subscription": "free" }
Response
201 Created 400 Bad Request 500 Server Error
{ "success": true, "message": "User registered successfully", "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "user": { "id": "507f1f77bcf86cd799439011", "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "country": "Nepal", "role": "user", "subscription": "free" } }
POST /api/auth/user/login
Login with email and password. Returns a JWT token for authenticated requests.
Request Body
Parameter Type Required Description
email String Yes User's email address

Response
200 OK 400 Bad Request 500 Server Error
{ "success": true, "message": "Login successful", "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "user": { "id": "507f1f77bcf86cd799439011", "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "country": "Nepal", "role": "user", "subscription": "free" } }

üó∫Ô∏è Trail & GeoJSON APIs

POST /api/trails
Create a new trail using a standard GeoJSON Feature.
Request Body
Parameter Type Required Description
type String Optional Must be "Feature" (default: "Feature")
geometry Object Yes GeoJSON geometry object (Point, LineString, Polygon, etc.)
properties Object Optional Arbitrary properties (e.g., name, difficulty, fid)
Request Example
{ "type": "Feature", "geometry": { "type": "LineString", "coordinates": [ [84.3294, 28.2096], [84.3500, 28.2200] ] }, "properties": { "name": "Annapurna Circuit", "difficulty": 5, "fid": 123 } }
Response
201 Created
POST /api/trails/feature-collection
Import multiple trails from a GeoJSON FeatureCollection.
Request Body
{ "type": "FeatureCollection", "features": [ { "type": "Feature", "geometry": { ... }, "properties": { ... } }, { "type": "Feature", "geometry": { ... }, "properties": { ... } } ] }
Response
201 Created
GET /api/trails
Get all trails with pagination and property filtering.
Query Parameters
Parameter Type Description
page Number Page number (default: 1)
limit Number Items per page (default: 100)
fid Number Filter by property 'fid'
name String Filter by property 'name' (regex)
difficulty Number Filter by property 'difficulty'
GET /api/trails/:id
Get a specific trail by its ID.
Path Parameters
Parameter Type Required Description
id String Yes Trail ID
Response
200 OK 404 Not Found 500 Server Error
GET /api/trails/near/point
Find trails near a specific point using geospatial queries.
Query Parameters
Parameter Type Required Description
longitude Number Yes Longitude (-180 to 180)
latitude Number Yes Latitude (-90 to 90)
maxDistance Number Optional Maximum distance in meters (default: 10000)
minDistance Number Optional Minimum distance in meters (default: 0)
limit Number Optional Maximum results (default: 50)
Request Example
GET /api/trails/near/point?longitude=86.9250&latitude=27.9881&maxDistance=5000
Response
200 OK 400 Bad Request 500 Server Error
POST /api/trails/within/polygon
Find trails within a specified polygon area.
Request Body
Parameter Type Required Description
polygon Object Yes GeoJSON Polygon object
Request Example
{ "polygon": { "type": "Polygon", "coordinates": [[ [86.9, 27.9], [87.0, 27.9], [87.0, 28.0], [86.9, 28.0], [86.9, 27.9] ]] } }
Response
200 OK 400 Bad Request 500 Server Error
GET /api/trails/within/radius
Find trails within a circular area (radius in kilometers).
Query Parameters
Parameter Type Required Description
longitude Number Yes Center longitude
latitude Number Yes Center latitude
radius Number Optional Radius in kilometers (default: 10)
limit Number Optional Maximum results (default: 50)
Request Example
GET /api/trails/within/radius?longitude=86.9250&latitude=27.9881&radius=25

üîê Authentication

JWT Token: After successful login/registration, you'll receive a JWT token. Include this token in the Authorization header for protected routes:
Authorization: Bearer <your-jwt-token>
Token Expiration: Tokens expire after 7 days (configurable via JWT_EXPIRES_IN environment variable).
Admin Routes: Admin APIs require both authentication (JWT token) and admin role. Use /api/admin/auth/login to login as admin. Admin accounts can exist in both User and Guide models.
Role-Based Access:
  • User routes: Available to all authenticated users
  • Guide routes: Available to guides and admins
  • Admin routes: Only accessible to users with admin role
